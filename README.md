# 🤖 AI Resume Analyzer

A 3-agent AI pipeline that automates the job application workflow —
upload your resume, paste a job description, see your match score,
and generate a tailored cover letter in seconds.

## ✨ How It Works
1. 📄 **Upload Your Resume** — Upload your resume in PDF format
2. 📋 **Paste Job Description** — Add the job description you are applying for
3. 🎯 **See Your Match Score** — Get a percentage match before committing to apply
4. ✉️ **Generate Cover Letter** — Receive a personalized cover letter tailored to the job

## 🛠️ Tech Stack
- **Python** — Core application logic
- **GROQ API** — Powers the multi-agent AI pipeline
- **Multi-Agent Architecture** — Separate agents for parsing, scoring, and writing
